## 🚀 Как запустить (3 простых шага)

### Шаг 1: Установка
```bash
cd Gryadka
pip install -r requirements.txt
```

### Шаг 2: Настройка
```bash
# Скопируйте .env.example в .env
cp .env.example .env

# Отредактируйте .env:
# - BOT_TOKEN - получите от @BotFather
# - ADMIN_ID - получите от @userinfobot
# - MINI_APP_URL - URL вашего Mini App
```

### Шаг 3: Запуск
```bash
# Инициализация БД с тестовыми данными
python init_db.py

# Запуск всех компонентов
python run.py
```

Готово! 🎉
